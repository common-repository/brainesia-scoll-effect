A plugin to make animation on your content when visitors scroll the website page.

== Description ==
Brainesia Scroll Effect will make animation on your content when visitors scroll the website page. 
You will see the Brainesia Scroll Effect icon in your editor. Just click the icon and you will see the box to choose an effect.

== Installation ==
1. Upload the entire `brainesia-scroll-effect` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
